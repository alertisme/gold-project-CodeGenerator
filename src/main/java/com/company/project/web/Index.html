FX168财经报社(香港)讯 周一（8月28日）美市盘中，美元延续低迷表现跌向92关口，现货黄金以及主要非美货币则强势上涨，其中现货黄金短线暴拉约10美元一举击穿1300美元/盎司大关。在美元跌跌不休之际，有市场人士表示，随着欧元和人民币等主要非美货币涨势如虹，美元的信誉正遭到质疑，其地位或遭到下调。

美元跌至新低 巨量买单涌现黄金站上千三关口

美市盘中，美元跌势加剧，美元指数一度低见92.24，为2016年5月初以来的最低水平。

（美指30分钟走势图，来源：FX168财经网）

澳洲西太平洋银行市场策略师Imre Speizer称，在德州引发洪灾的风暴哈维可能令美元进一步承压，因这是“严重的不利天气事件”，而且“显然对经济有害”。

欧元/美元周一在淡静交投中升至接近1.20美元的两年半高位，延续上周末尾的涨势，当时欧洲央行行长德拉基在杰克逊霍尔年会中并未放话打压欧元。

伦敦周一适逢公众假期，市场交投清淡，多数货币在窄幅区间交投。

美市盘中，欧元/美元上涨至1.1978。欧元/美元本月可望连续第六个月录得涨幅，为五年来最长连涨期。

（欧元/美元30分钟走势图，来源：FX168财经网）

欧元上周五上涨约1%，德拉基在杰克逊霍尔会议上的讲话聚焦于全球贸易等议题，并未提及欧元近期强势。

“这是因为他并未说什么，”德国商业银行外汇策略师Esther Reichelt表示，“他似乎不是特别担心欧元目前汇价，这正是市场推高欧元的理由。”

Reichelt表示，美联储主席耶伦在讲话中未提到美国货币政策，也鼓舞了欧元。市场目前预期到明年底之前只升息一次。

“市场的看法得到了证实，这支持美元再走软一点，”她补充说。

市场普遍预期欧元仍将持坚，至少短期来看如此，投资者焦点仍放在欧洲央行及其是否在9月政策会议上宣布缩减购债计划。

其他主要非美货币全线上扬：美元兑加元一度升破1.25关口，刷新8月25日以来高点，现报1.2493。美元兑墨西哥比索日内涨幅逾1%，现报17.8119，此前美国总统特朗普对北美自由贸易协定发表了评论。

在岸人民币突破6.61，离岸人民币突破6.62，均为2016年6月24日以来首次。

随着美元回落，现货黄金短线急涨约10美元，涨破1300美元关口，续刷2016年11月9日以来新高至1307.66美元/盎司。现货白银也短线上涨，日内涨近2%，刷新6月14日以来新高至17.41美元/盎司。

自上周五收盘以来，自飓风哈维展现最强破坏力之前，黄金是表现最佳的资产，力压债券和美元，美股也小幅走低...

(图片来源：Zerohedge、FX168财经网)

据报道，COMEX最活跃12月黄金期货合约在北京时间22:59-23:02四分钟成交量为15237手，交易合约总价值近20亿美元。

芝加哥大宗商品经纪商Linn Group策略师Ira Epstein表示，12月期金触及日内高点1313.30美元/盎司，为美国大选以来的最高水平，这是因为央行行长们无所作为的缘故。

Julius Baer分析师Carsten Menke称， “德拉基没有提到强势欧元是阻止货币政策正常化的因素，这引发了欧元反弹，这同时也反映到了黄金价格里。”

欧元和人民币强势上涨 美元地位岌岌可危？

周一，一位投资经理表示，随着欧元和人民币等竞对货币获得力度，美元的信誉遭到质疑。

尽管美国经济表现良好且美联储暗示将进一步加息，但投资者担忧这一政策将于何时、以何种方式祭出。

美国总统特朗普并未给出税改和基建支出这两大竞选承诺的具体细节。这一点再加上白宫丑闻迭出使得市场紧张不安，促使投资者抛弃美元转向其他币种。

“一方面，有一个缩表时机的短期讨论，另一方面则在讨论削减资产购买计划，”Carmignac投资委员会成员Didier Saint-George在谈及美联储下一步行动时表示。

“但我认为更重要的是，我提及的这一点正在发生，其是市场对货币的整体看法。从全球资产配置角度来看，美元的地位正在受到质疑。美元一定要证明其跟过去数年的地位相同吗？我认为战略上美元的地位或将被下调，这将令欧元受益，其很可能不仅令欧元受益，也将令人民币受益，”Saint-George说道。

Saint-George在谈及美元的涨势时表示，2014年美元交投于80附近，随后迅速攀升至100，2015年则一举触及100大关。尽管他预计美元指数不会跌回至80，但他并不排除其将自当前的92.425关口进一步回落的可能。

“我认为问题在于，在这些水平下美指将崩溃还是开始回落向2014年涨势之前的那种水平？”他称。

与此同时，周一欧元再次触及新高。欧元/美元攀升至两年半高位，因市场担忧美元，而欧洲央行行长德拉基上周在两次讲话中都未提及该行货币政策的下一步行动。

包括上周三公布的好于预期的PMI数据在内的积极经济数据以及一些欧元区经济体的大规模经常帐盈余使得投资者对买入欧元充满信心。

不过，欧元区面临一些政治风险。Wisdom Tree研究主管Viktor Nossek表示，2018年到来的意大利大选可能会成为欧元的问题。

“我们认为明年首个季度这可能会成为一个障碍，因失业率非常高，五星运动与民主党并驾齐驱，因此没有迹象显示意大利的政治风险已经减弱，”Nossek表示。

FX168财经报社(香港)讯 周一(8月28日)欧市盘中，现货黄金温和上涨，日内最高升至1299.40美元/盎司，逼近1300美元/盎司关口，且创下逾一周以来的最高水平，缘于美元下跌！

据彭博报道，黄金期货有望今年以来首次收于1300美元上方，因美联储在杰克逊霍尔年会上对政策问题的几近沉默激励了投资者，导致美元受挫而黄金需求升温。

金价今年以来已累计上涨13%。Comex 12月交割的黄金期货一度上涨0.5%至每盎司1304.50美元。

(黄金走势图 来源：FX168财经网、彭博)

尽管黄金期货今年曾在盘中交易时一度涨至1306.90美元，但自去年11月以来从未收盘成功站上1300美元关口。

黄金今年上涨，因投资者认为在低迷的通胀形势下美联储跟进加息的压力不大，而且总统特朗普飘忽不定的政策风格也激发了黄金需求。

此外，美国和朝鲜之间的紧张局势也有助于黄金走势。

美银美林上周表示，金价有望在明年初触及1400美元，因美国长期利率下降且特朗普在政策执行方面缺乏进展。

美联储(FED)主席耶伦上周五表示，2007-2009危机之后进行的改革增强了金融体系，没有妨碍经济成长；对于这些监管规则做出任何改变，都应该是小幅度的。

德拉基周五表示，欧洲央行超宽松货币政策正在发挥效果，而且欧元区经济复苏已站稳脚跟，即便还需要更多时间来提振通胀升至欧洲央行2%的目标水准。

“围绕杰克森霍尔的措辞一部分是乐观的，归功于欧洲央行行长称‘全球复苏正在转强’，同时欧元区增长‘提速’，”华侨银行分析师Barnabas Gan说。

Barnabas Gan还指出，“耶伦和德拉基对货币政策措施含糊其辞，从而给了市场人士在货币政策不确定之际推升黄金价格的理由。”

“若特朗普退出NAFTA的威胁和朝鲜紧张局势继续支撑避险需求，黄金价格本周可能突破每盎司1300美元，”Gan说。
